# Introduction

## 인생을 게임처럼!

스프린터는 AR을 기반으로 인생을 게임으로 바꿔주는 확장현실 서비스를 꿈꾸고 있습니다.

자비스처럼 눈 앞에 내 상태창이 뜨고, 목표를 위한 퀘스트가 제시되는 그런 세상 말입니다.

AR과 Gamification은 유저의 평범한 일상에 경이로움을 더해 성장경험을 극대화할 것입니다.
